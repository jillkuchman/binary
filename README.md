##Binary Converter

###Developers
Jill Kuchman & Conor Baumgart

###Date
09 April 2015

###Description
This app converts a binary input into its decimal value. The concepts used illustrate practice with BDD.

###Use and Editing
You can use and/or edit this application by:<br />
1. Cloning this project to the directory of your choice<br />
2. Opening the project in your favorite text-editor (optional)<br />
3. Open and point your browser to binary.html<br />
*(ex: file:///project-directory/binary.html)*
